My Projects

Fun things I’ve built/worked on!

  • Visualizing Epidemiological Rift-valley Analytics (VERA) - Fall 2025

    Collaborated with a team of four and a NASA domain expert to develop VERA, creating the website’s template UI foundation and implementing back-end query logic powering an interactive map & hover information display system for ~30 years of NASA RVF case data for Africa, Saudi Arabia, and Yemen.

  • Digits Data Classification Algorithms - Fall 2025

    Built an end-to-end digit-classification suite entirely from scratch—implementing 2D-Perceptron, Linear Regression, high-order Legendre models, k-NN with CV, RBF networks, neural networks, and SVMs using only NumPy and core Python.

  • Board Solver - Fall 2024

    Implemented a trie-based solving algorithm for Scrabble move generation within a computer-vision powered board game assistant mobile app optimized for runtime to search the official scrabble dictionary for possible moves.

    https://github.com/henryrobbinss/BoardSolverMobile

  • AccessACity - HackRPI 2024

    Winner of 'Ship Fast, Ship Often - Build a Payworthy MVP in a Weekend' prize, AccessACity provides users with a network of events labelled for their accessibility. Worked on map functionality and display in a 4-person team.

    https://devpost.com/software/accessacity

  • CentTrack - HackRPI 2023

    Winner of the 'Best Mobile Hack' prize, CentTrack is a budget tracket to help users organize their spendings, earnings, and savings based on a chosen plan. Worked on front-end development and graphics in a 4-person team.

    https://devpost.com/software/track-xnat9j
